# ConnectaTel Professional Analysis 
This repository contains the comprehensive analysis performed for the Sprint 7 Capstone Project - ConnectaTel Case Study.

The `ConnectaTel case` dataset includes 40,000 customer records featuring missing values, sentinel values, outliers, and real-world data quality challenges.
## 📂 Repository Content

- `notebooks/Copia_de_S7_Project_ConnectaTel.ipynb`
  → The primary notebook featuring data cleaning, Exploratory Data Analysis (EDA), distribution modeling, outlier detection, and professional executive summaries.

1. Open `notebooks/ConnectaTel.ipynb`
2. Execute the cells in sequiential order
3. The notebook automatically loads the dataset from the /data/ directory or via a public link (as configured).

## 🧠 Project Objectives

- Explore, clean, and analyze this data to build a statistical profile of customers, detect atypical behaviors, and create customer segments.
- Identify consumption patterns, design retention strategies, and suggest improvements to the plans offered by the company.
- Generate actionable business recommendations for ConnectaTel’s Strategy and Integration team.
